# react-hoos-bigdata

vite+react+hooks 实现 10 万级数据表格渲染
之前使用 vue 实现了一版，这次利用 react 实现。

## 下载

> git clone https://github.com/tcly861204/vite-react-hooks-bigdata.git

## 安装

> npm install

## 运行

> npm run dev

## 打包

> npm run build
